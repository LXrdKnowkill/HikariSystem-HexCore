## Quick Start

1. Download the latest release from [Releases](https://github.com/LXrdKnowkill/HikariSystem-HexCore/releases)
2. Extract the archive
3. Run `HexCore.exe` (Windows) or `./hexcore` (Linux)
4. Right-click any file to access HexCore analysis tools

---

## Features Overview

| Feature | Description |
|---------|-------------|
| **Hex Viewer** | View and analyze binary files with virtual scrolling |
| **PE Analyzer** | Comprehensive Windows executable analysis |
| **Hash Calculator** | Calculate MD5, SHA-1, SHA-256, SHA-512 |
| **Strings Extractor** | Extract and categorize ASCII/Unicode strings |
| **Entropy Analyzer** | Visual entropy graph for packer detection |
| **Base64 Decoder** | Find and decode Base64 encoded data |
| **File Type Detector** | Identify true file type via magic bytes |

## System Requirements

- **Windows**: Windows 10/11 (64-bit)
- **Linux**: Ubuntu 20.04+ or equivalent (64-bit)
- **RAM**: 4GB minimum, 8GB recommended
- **Disk**: 500MB for installation
