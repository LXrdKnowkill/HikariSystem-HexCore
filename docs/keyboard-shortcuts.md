# Keyboard Shortcuts

Productivity shortcuts for HexCore IDE.

---

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl+O` | Open file |
| `Ctrl+Shift+P` | Command Palette |
| `Ctrl+,` | Settings |
| `Ctrl+` ` | Toggle Terminal |
| `Ctrl+B` | Toggle Sidebar |

---

## Hex Viewer

| Shortcut | Action |
|----------|--------|
| `Ctrl+G` | Go to Offset |
| `Ctrl+F` | Search Hex Pattern |
| `Ctrl+C` | Copy Selection |
| `Page Up/Down` | Scroll viewport |
| `Home` | Go to start |
| `End` | Go to end |

---

## File Explorer Context Menu

Right-click any file to access:

| Command | Description |
|---------|-------------|
| HexCore: Open Hex View | Binary viewer |
| HexCore: Analyze PE File | PE analysis (exe/dll) |
| HexCore: Calculate File Hashes | Hash calculation |
| HexCore: Quick Hash (SHA-256) | Fast SHA-256 |
| HexCore: Verify Hash | Compare hashes |
| HexCore: Extract Strings | String extraction |
| HexCore: Entropy Graph | Entropy analysis |
| HexCore: Decode Base64 | Base64 decoder |
| HexCore: Detect File Type | Magic bytes |

---

## Command Palette

Access all HexCore commands via `Ctrl+Shift+P`:

1. Press `Ctrl+Shift+P`
2. Type `HexCore`
3. Select command from list
